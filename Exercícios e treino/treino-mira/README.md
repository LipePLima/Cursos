# Treino-mira
## Descrição
 Projeto simples, feito para treino de mira(jogo Counter-Strike Global Offensive). Feito com a plataforma Alura. 
 
 Para jogar, basta clicar no link e já irá começar com um alvo pequeno aparecendo em várias posições pela tela. O desafio é clicar no alvo e irá aparecer uma janela de diálogo informando que você acertou. Basta clicar em "continuar" que o alvo volta a se movimentar
